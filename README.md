# Project Night-Ops: Sleep Performance Ecosystem

Project Night-Ops is an influencer-driven sleep optimization platform that combines wearable data aggregation with a social "Squad" leaderboard and a commerce layer for performance-grade supplements.

An iOS-exclusive mobile application built with Expo and React Native.

### Key Features
- **The Squad (Leaderboard)**: Social accountability through asynchronous daily sleep score syncs.
- **Performance Readiness Score**: Aggregated scores from Apple HealthKit and Oura.
- **Retrieval-Based AI**: A sleep chatbot grounded in peer-reviewed studies and specific influencer protocols (e.g., Huberman Lab).
- **Recovery Protocol**: Non-prescriptive, study-backed recommendations based on recovery metrics.
